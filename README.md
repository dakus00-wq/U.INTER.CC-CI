# Site Web UCCC

Site web moderne et responsive pour l'Union des Coopératives Café Cacao de Côte d'Ivoire.

## Utilisation

Ouvrez le fichier `index.html` dans votre navigateur web pour voir le site.

## Sections

- **Accueil** : Page d'accueil avec introduction.
- **À propos** : Histoire et mission de l'UCCC.
- **Nos Coopératives** : Informations sur les coopératives membres.
- **Production Café & Cacao** : Détails sur les processus de production.
- **Actualités** : Nouvelles récentes.
- **Contact** : Formulaire de contact et informations.

## Technologies

- HTML5
- CSS3 (avec Bootstrap 5)
- JavaScript

## Développement

Pour modifier le site, éditez les fichiers suivants :
- `index.html` : Structure et contenu.
- `style.css` : Styles personnalisés.
- `script.js` : Interactivité.

## Dépannage

- Assurez-vous d'avoir une connexion internet pour charger Bootstrap depuis le CDN.
- Si le site ne s'affiche pas correctement, vérifiez la console du navigateur pour les erreurs.
